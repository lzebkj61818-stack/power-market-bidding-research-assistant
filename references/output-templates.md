# Output Templates

Use these templates when the user wants a stable memo format.

## Report Interpretation Memo

```text
Market Context

Known Facts

Assumptions

Interpretation

Possible Impact On Next Session

Key Risks

Next Checks
```

## Strategy Audit Memo

```text
Market Context

Observed Facts

Audit Findings

Timing Or Visibility Risks

Action Mapping Risks

What Can Be Concluded

What Cannot Be Concluded
```

## Backtest Review Memo

```text
Backtest Scope

Known Facts

Strengths

Weaknesses

Leakage Or Timing Concerns

Decision

Next Checks
```

## Candidate Strategy Memo

```text
Market Context

Known Facts

Assumptions

Candidate Strategy A

Candidate Strategy B

Key Risks

Human Decision Points
```
